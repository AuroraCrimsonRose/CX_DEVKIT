# CXEX Dev Studio

CXEX Dev Studio is an internal development environment and tooling suite for building, testing, and inspecting CXEX-based systems within the Catalyst Labs ecosystem.

It provides utilities for working with low-level CX formats, experimental runtime systems, and related development tooling.

---

## Purpose

CXEX Dev Studio is used for developing and maintaining CX ecosystem components, including:

- CXEX binary formats and tooling
- CXK / CXOS development workflows
- Experimental ABI and IPC systems
- Prototype runtime environments
- Internal debugging and inspection tools

It is intended for developer use only and is tightly coupled with ongoing CX ecosystem development.

---

## Features

- CXEX format parsing and inspection tools
- Internal build and runtime utilities
- Debugging tools for CX-based systems
- Experimental development helpers
- Lightweight project scaffolding for rapid iteration

---

## Status

This project is an internal development tool.

It is actively evolving alongside CX ecosystem components and may change without stability or backward compatibility guarantees.
